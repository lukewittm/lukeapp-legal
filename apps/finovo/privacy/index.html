Datenspeicherung & Löschung

4.1 Lokale Speicherung

Alle Daten werden primär lokal auf Ihrem iOS-Gerät gespeichert (SwiftData, UserDefaults, Keychain für sensible Flags). Durch Deinstallation der App werden alle lokalen Daten entfernt. Die App bietet zusätzlich eine „Daten zurücksetzen"-Funktion in den Einstellungen.

4.2 iCloud-Speicherung (CloudKit)

Sofern Sie iCloud verwenden, werden Ihre Daten über Apple CloudKit in Ihrer eigenen privaten iCloud-Datenbank gespeichert. Der Entwickler hat keinen Zugriff auf diese Daten. iCloud-Daten können über Einstellungen → [Ihr Name] → iCloud → Speicher verwalten → Finovo entfernt werden.

4.3 Löschmarker („Tombstones")

Beim Löschen eines Eintrags wird kurzzeitig ein Löschmarker aufbewahrt, damit die Löschung auch auf Ihre weiteren Geräte übertragen wird. Anschließend wird er automatisch komprimiert entfernt.

4.4 Speicherdauer

Daten werden für die gesamte Nutzungsdauer gespeichert und auf Wunsch sofort gelöscht. Es bestehen keine gesetzlichen Aufbewahrungspflichten für die innerhalb der App gespeicherten Haushaltsdaten.

6

Freigabe von Daten an einen Partner

Wenn Sie den Partner-Modus aktivieren und eine Freigabe erzeugen, werden ausgewählte Haushaltsdaten über Apple CloudKit Sharing (CKShare) mit dem iCloud-Konto der eingeladenen Person geteilt. Dabei gilt:

  • Sie kontrollieren vollständig, ob und mit wem Sie teilen. Die Freigabe erfolgt aktiv über die iOS-Systemdialoge.
  • Sie können die Freigabe jederzeit widerrufen; die weitere Synchronisation geteilter Inhalte endet dann.
  • Die Datenübertragung ist TLS-verschlüsselt und läuft ausschließlich über Apples iCloud-Infrastruktur.
  • Einkommensdaten werden nicht automatisch geteilt – sie dienen nur zur lokalen Berechnung von Aufteilungsverhältnissen.

4

Storage & deletion

4.1 Local storage

All data is primarily stored on your iOS device (SwiftData, UserDefaults, Keychain for sensitive flags). Uninstalling the App removes all local data. The App additionally provides a "Reset data" option in Settings.

4.2 iCloud storage (CloudKit)

If you use iCloud, your data is stored via Apple CloudKit in your own private iCloud database. The developer has no access to this data. iCloud data can be removed via Settings → [your name] → iCloud → Manage Storage → Finovo.

4.3 Deletion markers (tombstones)

When you delete a record, a tombstone is briefly retained so the deletion propagates to your other devices, after which it is automatically compacted away.

4.4 Retention

Data is retained for the entire duration of use and deleted on request without delay. There are no statutory retention obligations for the household data stored in the App.

Sharing data with a partner

When you enable Partner Mode and create a share, selected household data is shared via Apple CloudKit Sharing (CKShare) with the iCloud account of the invited person. The following applies:

  • You have full control over whether and with whom you share. Sharing is initiated actively via iOS system dialogs.
  • You can revoke the share at any time; further synchronisation of shared content then ends.
  • Data transmission is TLS-encrypted and takes place exclusively through Apple's iCloud infrastructure.
  • Income data is not shared automatically — it is used only locally to calculate split ratios.
